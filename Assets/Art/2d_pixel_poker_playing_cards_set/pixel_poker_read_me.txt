About
-----
Play cards, poker and dice with these game assets.


Contents
--------
Playing Cards with multiple backs @ 40 x 56 pixels.
Poker Chips @ 8/16/24/32 pixels.
Dice @ 8/16/24/32 pixels.
UI outline selection & pointer graphics.


Format
------
All in png format and sharing one 16-colour palette.
Aseprite format files for the above.
